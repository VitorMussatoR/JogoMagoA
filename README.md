Integrantes:

## Descrição do Projeto:

Magoraios é um jogo em primeira pessoa desenvolvido na Unity. O jogador controla um mago capaz de utilizar ataques mágicos para eliminar inimigos enquanto atravessa uma arena repleta de obstáculos e desafios. O objetivo é sobreviver aos inimigos, evitar armadilhas e utilizar suas habilidades para avançar pelo cenário.
